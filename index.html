Streamy by daio

Streamy is a SIMBL plugin for QuickTime X, that lets you enable and disable video, audio and other streams on the fly.

Install

  1. Get the latest version of Streamy from here.
  2. If you don't yet have SIMBL, install it from here.
  3. Quit QuickTime X
  4. Put Streamy.bundle into Library/Application Support/SIMBL/Plugins/ either in your home folder or in the root.
  5. Launch QuickTime X, you should see Streamy menu bar
